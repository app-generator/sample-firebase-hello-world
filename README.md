# firebase-example:
1- Download the code and open it to VS code editor
2- Setup Firebase and update "firebaseConfig"  value in index.html

     var firebaseConfig = {
        apiKey: "",
    authDomain: "",
    projectId: "",
    storageBucket: "",  
    messagingSenderId: "",
    appId: ""
      };

2- Go to View --> Extensions option and find "Live Server" and install it
3- Right click on "index.html" and choose option "Open with live server"
4- A browser will open showing front-end of web app with Google Authentication button.
